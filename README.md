# OWNYOURWEB AI-Powered Marketing System

A polished, responsive landing page for the OWNYOURWEB Systems AI-powered marketing offer at one price: $250 per month.

## Live site

The site is published with GitHub Pages from the repository root.

## Local preview

Open `index.html` in a browser or run a simple static server from this folder:

```bash
python3 -m http.server 4173
```

Then visit `http://localhost:4173`.

## Lead form

The application form submits directly to the configured Formspree intake endpoint, so visitors can apply without opening an email app.
