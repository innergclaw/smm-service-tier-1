# Own Your Web Paid Social Campaign

Production-ready campaign package for Facebook, Instagram, and TikTok.

## Start here

- `campaign-playbook.md` — campaign structure, targeting, testing, measurement, and launch checklist.
- `ad-copy.md` — Meta copy, TikTok captions, retargeting copy, vertical hooks, and comment replies.
- `voiceover-scripts.md` — three 15-second scripts, one 30-second script, hooks, and a text-led shot sequence.
- `visual-philosophy.md` — the creative system behind the campaign.
- `images/` — nine ready-to-upload PNG ads.

## Image map

Each of the three concepts includes:

- `feed-4x5` — 1080 × 1350 for Facebook and Instagram feeds.
- `square-1x1` — 1080 × 1080 for flexible feed placements.
- `vertical-9x16` — 1080 × 1920 for Instagram Stories/Reels and TikTok.

Concepts:

1. `01-system` — system/consistency message.
2. `02-stop-scrambling` — pain-to-relief message.
3. `03-one-price` — transparent $450 offer.

## Publishing notes

The PNGs are static placements. For voiceover ads, use the 9:16 design as the visual foundation, animate the text/card sequence described in `voiceover-scripts.md`, and add a commercially licensed AI voice. TikTok content containing AI-generated audio should be disclosed in accordance with current platform policy.

All calls to action should point to the application section on the live landing page. Add platform- and creative-level UTMs before publishing.
